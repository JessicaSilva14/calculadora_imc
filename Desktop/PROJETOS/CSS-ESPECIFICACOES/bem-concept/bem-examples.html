Exemplos Práticos da Metodologia BEM

Abaixo você verá componentes reais implementados seguindo rigorosamente a metodologia BEM, com blocos, elementos e modificadores.

1. Header com Navegação

Minha Loja
Home Produtos Sobre Contato

Código BEM do Header:

<div class="header"> <div class="header__logo">Minha Loja</div> <nav class="header__nav"> <a href="#" class="header__nav-link header__nav-link--active">Home</a> <a href="#" class="header__nav-link">Produtos</a> <a href="#" class="header__nav-link">Sobre</a> <a href="#" class="header__nav-link">Contato</a> </nav> </div>

Benefícios Demonstrados

✅ Componentes Reutilizáveis

Os blocos product-card, header e search-form podem ser reutilizados em qualquer página.

✅ Modificadores Flexíveis

Modificadores como --featured, --out-of-stock alteram aparência sem duplicar código.

✅ Estrutura Clara

Nomes de classes auto-explicativos facilitam manutenção e colaboração.

✅ Escopo Isolado

Cada bloco tem seu próprio escopo, evitando conflitos de CSS.
