📁 Arquitetura SMACSS

BASE: Reset e estilos padrão
LAYOUT: Grid e containers responsivos
MODULE: Componentes BEM (produto, elementos, modificadores)
STATE: Estados (is-active, is-disabled)
THEME: Tema A (quente) e Tema B (elegante) com paletas personalizadas
