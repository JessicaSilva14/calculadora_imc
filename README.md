# ⚖️ Calculadora de IMC Pro

Uma calculadora de Índice de Massa Corporal (IMC) desenvolvida com **React** e **Vite**.

## ✨ Funcionalidades
- Cálculo de IMC em tempo real (onChange).
- Tabela de classificação dinâmica com destaque visual.
- Histórico de resultados persistente (salvo no LocalStorage do navegador).
- Interface responsiva com SASS.

## 🚀 Tecnologias Utilizadas
- [React 19](https://react.dev/)
- [Vite 8](https://vitejs.dev/)
- [SASS/SCSS](https://sass-lang.com/)
- [JavaScript ES6+](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript)

## 📦 Como rodar o projeto
1. Clone o repositório.
2. Instale as dependências com `npm install`.
3. Inicie o servidor de desenvolvimento com `npm run dev`.
